### 문제 설명

개미집을 건설하고 철거하며, M마리의 개미가 모든 개미집을 정찰할 수 있는 최소 시간을 구하는 문제이다.

### 입력

* 첫 줄에 쿼리 개수 Q가 주어진다.
* 이후 Q개의 줄에 다음 네 가지 명령 중 하나가 주어진다.

  * `100 N x1 x2 ... xN` : 마을 건설
  * `200 x` : 개미집 건설
  * `300 q` : q번째 개미집 철거
  * `400 M` : 개미 M마리로 정찰 시 필요한 최소 시간 출력

### 출력

* 명령이 400인 경우마다 최소 시간을 한 줄에 하나씩 출력한다.
