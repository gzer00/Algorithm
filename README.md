# 🧠 알고리즘 스터디

> 매주 수요일에 만나 코드 리뷰를 진행하는 알고리즘 스터디입니다.
> 각자 정해진 문제를 풀고, 자신의 이름 폴더에 업로드합니다.

---

## 📅 스터디 진행 방식

1. 매주 문제를 선정하고, 그 주차 폴더(`Week01`, `Week02`, ...)를 생성합니다.
2. 각자 자신의 이름 폴더에 문제 풀이 코드를 업로드합니다.
3. 필요시 각 문제마다 간단한 설명과 복잡도를 `README.md`에 정리합니다.
4. 매주 수요일 코드 리뷰를 진행하며 서로 피드백을 주고받습니다.

## 📌 파일 및 폴더 네이밍 규칙

| 항목 | 규칙 | 예시 |
|------|------|------|
| 주차 폴더 | `Week01`, `Week02`, ... | `Week05` |
| 이름 폴더 | 영어 이름 또는 GitHub ID | `minwook`, `jiwon` |
| 파일 이름 | `[플랫폼]_[문제번호].[확장자]` | `BOJ_1000.py`, `PGS_42576.java` |

### 플랫폼 약어 예시
- `BOJ` : 백준 (Baekjoon Online Judge)
- `PGS` : 프로그래머스 (Programmers)
- `SWEA` : 삼성 SW Expert Academy
- `LTC` : LeetCode
- `CFC` : Codeforces
